# ScroogeLoot
A loot system addon created for Epoch WoW, for the guild Disco Ducks. Originally created for the guild Dancing Boars on Turtle WoW. Same creators.

Author: Vim (from Epoch WoW)
Brainstormers: Albin, Arkintos and Markikus. (from Epoch WoW)

If you do not know the installation process, you may need to start with a simpler addon that this lol. 

This system is fairly complex but still as fair as we could muster in a game like this. Those that still commit a lot of time to your guild still gets advantage over the newcomers, but newcomers still have a chance at really good loot. Thus was this addon created. We used to do this manually on google sheets, but it got too annoying to deal with so I decided to start coding again, after a 7 year hiatus. Sure, I've cut some corners with AI, here and there, but hinestly, I don't see that as a problem but just a tool.
